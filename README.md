# Retail-Sales-Analysis
The project demonstrates how spreadsheet tools and business intelligence software can be used to transform raw retail sales data into clear visual insights that support data-driven decision making.
The project analyzes retail sales data to identify trends, product performance, and overall sales patterns. The objective is to transform raw sales data into meaningful insights that can help businesses understand their sales performance and make better decisions.
The dataset was cleaned and organized using Microsoft Excel, and interactive dashboards were created in Microsoft Power BI to visualize key sales insights.

Objectives:
Clean and organize retail sales data
Identify overall sales trends
Analyze product category performance
Visualize insights through interactive dashboards

Tools Used:
Microsoft Excel – Data cleaning and preparation
Microsoft Power BI – Data visualization and dashboard creation

Key Analysis:
The analysis focuses on:
Sales performance over time
Comparison of different product categories
Identifying high-performing products
Visualizing key metrics using dashboards
